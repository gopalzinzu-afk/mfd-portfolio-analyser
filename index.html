How it works
01
File reading
FileReader converts your .xlsx / .pdf to base64 in-browser. Nothing touches a server until the API call.
02
Claude analyses
Your file + full MFD system prompt sent to claude-sonnet-4-6. Rolling returns, tax, overlap, actions all analysed.
03
Structured JSON
Claude returns a clean JSON object with all six sections. The app validates and maps it to the workbook schema.
04
Excel export
SheetJS writes a multi-sheet .xlsx with colour-coded sections. Browser download — no upload needed.
